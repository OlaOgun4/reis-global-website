## Recommended deployment
Keep the domain registered at GoDaddy. Host this site on:
1. Cloudflare Pages
2. Vercel
3. Netlify
4. GoDaddy static/shared hosting

For Cloudflare Pages / Vercel / Netlify, upload or connect the folder and then point the GoDaddy DNS records to the host using the records supplied by the platform.
